template<typename Type>
btn::span class

Refers to a contiguous sequence of elements with the first element of the sequence at position zero.

Template parameters
Type Element type.

Contents

  • Reference
    • Public types
    • Constructors, destructors, conversion operators

The elements are not copied but referenced, so they should outlive the span to avoid dangling references.
