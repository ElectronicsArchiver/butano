btn::sprite_font class

Contains the required information to generate text sprites.

Contents

  • Reference
    • Public static variables
    • Constructors, destructors, conversion operators
    • Public functions

Currently, it supports 8x8 and 8x16 fixed width AND variable width characters of 16 colors (4 bits per pixel).

Also, UTF-8 characters are supported.

UTF-8 characters and character widths are not copied but referenced, so they should outlive the sprite_font to avoid dangling references.
