void* btn::memory::ewram_alloc(int bytes)

Allocates uninitialized storage in EWRAM.

Parameters
bytes Bytes to allocate.
Returns On success, returns the pointer to the beginning of newly allocated memory. To avoid a memory leak, the returned pointer must be deallocated with ewram_free. On failure, returns a null pointer.

void btn::memory::ewram_free(void* ptr)

Deallocates the space previously allocated by ewram_alloc.

Parameters
ptr Pointer to the memory to deallocate (it can be null).

template<typename Type>
void btn::memory::copy(const Type& source_ref, int elements, Type& destination_ref)

Copies the given amount of elements from the object referenced by source_ref to the object referenced to by destination_ref.

Parameters
source_ref Const reference to the memory location to copy from.
elements Number of elements to copy (not bytes).
destination_ref Reference to the memory location to copy to.

If the objects overlap, the behavior is undefined.

void btn::memory::set_bytes(uint8_t value, int bytes, void* destination_ptr)

Copies the given value into each of the first bytes of the object pointed to by destination_ptr.

Parameters
value Value to copy.
bytes Number of bytes to fill.
destination_ptr Pointer to the object to fill.

If the given size is greater than the size of the object pointed to by destination_ptr or it is not trivial, the behavior is undefined.
